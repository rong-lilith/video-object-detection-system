# 视频目标检测系统

## 项目简介

面向实时视频流或离线视频文件的目标检测系统，支持视频解码、目标识别、跟踪、结果保存与可视化展示。

## 核心功能

- 视频流读取与解码
- 目标检测模型推理
- 多目标跟踪
- 检测结果渲染
- 结果导出与接口服务
- 实时性能优化

## 技术栈规划

- Python
- OpenCV
- PyTorch / YOLO
- ONNX Runtime / TensorRT
- FastAPI
- FFmpeg
